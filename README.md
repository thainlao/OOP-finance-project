# Система управления личными финансами

Backend-приложение для управления личными финансами с CLI-интерфейсом.

## Функциональность

- ✅ Авторизация пользователей
- ✅ Учет доходов и расходов
- ✅ Установка бюджетов по категориям
- ✅ Статистика и аналитика
- ✅ Оповещения о превышении бюджетов
- ✅ Сохранение данных в JSON
- ✅ Валидация ввода
- ✅ Экспорт данных

## Установка и запуск

### Требования
- Java 11+
- Maven 3.6+

# Сборка проекта
mvn clean compile

# Запуск приложения
mvn exec:java -Dexec.mainClass="com.finance.Main"

# Запуск тестов
mvn test

# Создание JAR
mvn package
